# postgres-database-quiz-notes

## Quiz Questions

After completing this exercise, you should be able to discuss or answer the following questions:

- What is a database schema?
  a schema is a collection of tables and defines how data should be organized in a database.
- What is a table?
  a table is a set of rows with the same attributes, like an excel file.
- What is a row?
  a row is a complete entry in a table. that means its an id with a set of the attributes defined by the table.
- What is an attribute and what other names are used to describe them?
  an attribute is a value associated with a "column" for a row, more or less.
